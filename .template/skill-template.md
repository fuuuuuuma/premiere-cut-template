# スキルテンプレート

壁打きの回答を元に、以下の {{変数}} を埋めてスキルファイルを生成してください。
生成したファイルは `~/.claude/commands/{{COMMAND_NAME}}.md` に保存する。

## 変数一覧

| 変数 | 対応する質問 |
|---|---|
| `{{COMMAND_NAME}}` | Q9 コマンド名 |
| `{{MAIN_TRACK}}` | Q1 メイン音声トラック |
| `{{INPUT_DIR}}` | Q6 XMLの書き出し場所 |
| `{{OUTPUT_DIR}}` | Q7 カット済みXMLの保存先 |
| `{{SCRIPT_PATH}}` | Q8 silence_cut.py のパス |
| `{{THRESHOLD}}` | Q5 無音判定閾値（dB） |
| `{{MIN_SILENCE}}` | Q3 最小無音長（秒） |
| `{{PADDING}}` | Q4 前後余白（フレーム） |
| `{{OUTPUT_SUFFIX}}` | Q10 ファイル名末尾 |

---

## 生成するスキルファイルの内容

```
# Premiere Pro XML 無音カット

## Gotchas

- **{{MAIN_TRACK}} 以外のトラックで無音検出してしまう** → 必ず {{MAIN_TRACK}} で検出
- **Bashタイムアウトを指定し忘れる** → 必ず `600000`（10分）を指定

## 入力

- 引数にXMLファイルのパスを渡す
- 引数がない場合は会話の中で言及されたXMLパスを特定する。見つからなければユーザーに聞く

## 重要ルール

1. 無音検出は {{MAIN_TRACK}} トラックの音声で行う
2. V1 / A1 / A2 等、全トラックに同じタイムライン位置でカットを入れる
3. 各トラックのソースオフセットを個別に保持する

## 実行

確認不要。即実行する。Bashのタイムアウトは必ず **600000** を指定すること。

\`\`\`bash
python3 "{{SCRIPT_PATH}}" \
  "<XMLファイルの絶対パス>" \
  --output-dir "{{OUTPUT_DIR}}" \
  --threshold {{THRESHOLD}} \
  --min-silence {{MIN_SILENCE}} \
  --padding {{PADDING}}
\`\`\`

## 結果報告

\`\`\`
■ 無音カット結果
─────────────────────────
元の長さ:      XX分XX秒
カット後:      XX分XX秒
カットした無音: XX分XX秒
削減率:        XX.X%
出力ファイル:  <絶対パス>
─────────────────────────
\`\`\`

出力ファイル名: `<入力ファイル名>{{OUTPUT_SUFFIX}}.xml`
Premiere Pro で「ファイル > 読み込み」で読み込める旨を添える。
```
